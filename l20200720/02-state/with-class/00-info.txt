props  
-----
allow you to pass data from a parent 
(wrapping) component to a child (embedded) component.



state
------
Whilst props allow you to pass data down the component tree (and hence trigger an UI update),
state is used to change the component, well, state from within. 
Changes to state also trigger an UI update.


events
